# User Logout

Invalidate the current user's tokens and end the session.

## Endpoint

```
POST /api/v1/auth/logout
```

## Description

Logs out the authenticated user by revoking their refresh token. The access token remains valid until expiration.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token format: `Bearer {accessToken}`
</ParamField>

## Request Body

```json
{}
```

## Response

### Success (200)

```json
{
  "success": true,
  "message": "Logged out successfully"
}
```

### No Token (401)

```json
{
  "success": false,
  "message": "No authentication token provided",
  "code": "NO_TOKEN"
}
```

## Examples

<Tabs>
  <Tab title="cURL">
    ```bash
    curl -X POST http://localhost:3000/api/v1/auth/logout \
      -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
    ```
  </Tab>

  <Tab title="JavaScript">
    ```javascript
    const accessToken = localStorage.getItem('accessToken');

    const response = await fetch('http://localhost:3000/api/v1/auth/logout', {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${accessToken}`
      }
    });

    const data = await response.json();
    
    // Clear local storage
    localStorage.removeItem('accessToken');
    localStorage.removeItem('refreshToken');
    ```
  </Tab>
</Tabs>

## Security Notes

- Logout invalidates the refresh token immediately
- Access token remains valid until expiration (1 hour)
- For additional security, also clear client-side tokens
- Implement logout on all active sessions for complete logout
