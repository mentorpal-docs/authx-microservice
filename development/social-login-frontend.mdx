## Frontend Social Login Implementation Guide

This guide shows you how to implement social OAuth login on the frontend using React, Vue 3, and vanilla JavaScript.

## Prerequisites

- Backend OAuth endpoints running (see API Reference)
- OAuth credentials configured (Google, Facebook, GitHub)
- Node.js and npm/yarn installed
- Basic understanding of authentication flows

## Architecture Overview

```
┌─────────────────────────────────────────────────────┐
│              Frontend Application                     │
│  (React / Vue / Vanilla JS)                          │
└─────────────────────────────────────────────────────┘
                        ↓
        User clicks "Login with Google"
                        ↓
┌─────────────────────────────────────────────────────┐
│          Backend OAuth Endpoints                      │
│  GET /api/v1/auth/google                            │
│  GET /api/v1/auth/facebook                          │
│  GET /api/v1/auth/github                            │
└─────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────┐
│       OAuth Provider (Google/Facebook/GitHub)        │
│              User Authorization                      │
└─────────────────────────────────────────────────────┘
                        ↓
         Redirect back to /callback
                        ↓
┌─────────────────────────────────────────────────────┐
│     Backend Processes Callback & Returns JWT         │
└─────────────────────────────────────────────────────┘
                        ↓
         Frontend stores JWT & redirects home
```

### 2. OAuth Callback Handler Component

Create a callback page to handle redirects from OAuth providers:

```jsx
// pages/auth/OAuthCallback.jsx
import React, { useEffect, useState } from 'react';
import { useNavigate, useSearchParams } from 'react-router-dom';
import { useAuth } from '../../context/AuthContext';

function OAuthCallback() {
  const navigate = useNavigate();
  const [searchParams] = useSearchParams();
  const { setUser, setToken } = useAuth();
  const [error, setError] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    const token = searchParams.get('token');
    const error = searchParams.get('error');

    if (error) {
      setError(decodeURIComponent(error));
      setLoading(false);
      return;
    }

    if (token) {
      try {
        // Store token
        localStorage.setItem('accessToken', token);
        setToken(token);

        // Fetch user profile
        fetchUserProfile(token);
      } catch (err) {
        setError('Failed to process login');
        setLoading(false);
      }
    }
  }, [searchParams]);

  const fetchUserProfile = async (token) => {
    try {
      const response = await fetch(
        'http://localhost:3000/api/v1/user/profile',
        {
          headers: {
            Authorization: `Bearer ${token}`
          }
        }
      );

      if (response.ok) {
        const userData = await response.json();
        setUser(userData.data);
        navigate('/dashboard');
      } else {
        setError('Failed to fetch user profile');
      }
    } catch (err) {
      setError('An error occurred during login');
    } finally {
      setLoading(false);
    }
  };

  if (loading) {
    return <div className="loading">Processing your login...</div>;
  }

  if (error) {
    return (
      <div className="error-container">
        <h2>Login Error</h2>
        <p>{error}</p>
        <a href="/login">Back to Login</a>
      </div>
    );
  }

  return null;
}

export default OAuthCallback;
```

### 5. Auth Context for State Management

```jsx
// context/AuthContext.jsx
import React, { createContext, useContext, useState, useEffect } from 'react';

const AuthContext = createContext();

export function AuthProvider({ children }) {
  const [user, setUser] = useState(null);
  const [token, setToken] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    // Check if token exists in localStorage
    const storedToken = localStorage.getItem('accessToken');
    if (storedToken) {
      setToken(storedToken);
      // Optionally fetch user data
      fetchUserProfile(storedToken);
    } else {
      setLoading(false);
    }
  }, []);

  const fetchUserProfile = async (token) => {
    try {
      const response = await fetch(
        'http://localhost:3000/api/v1/user/profile',
        {
          headers: { Authorization: `Bearer ${token}` }
        }
      );
      if (response.ok) {
        const userData = await response.json();
        setUser(userData.data);
      }
    } catch (error) {
      console.error('Failed to fetch user profile:', error);
    } finally {
      setLoading(false);
    }
  };

  const logout = () => {
    localStorage.removeItem('accessToken');
    setUser(null);
    setToken(null);
  };

  return (
    <AuthContext.Provider value={{ user, token, loading, setUser, setToken, logout }}>
      {children}
    </AuthContext.Provider>
  );
}

export function useAuth() {
  const context = useContext(AuthContext);
  if (!context) {
    throw new Error('useAuth must be used within AuthProvider');
  }
  return context;
}
```

### 7. Account Settings with Linked Accounts

```jsx
// pages/AccountSettings.jsx
import React, { useEffect, useState } from 'react';
import { useAuth } from '../context/AuthContext';

function AccountSettings() {
  const { token } = useAuth();
  const [linkedAccounts, setLinkedAccounts] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    fetchLinkedAccounts();
  }, []);

  const fetchLinkedAccounts = async () => {
    try {
      const response = await fetch(
        'http://localhost:3000/api/v1/auth/linked-accounts',
        {
          headers: { Authorization: `Bearer ${token}` }
        }
      );

      if (response.ok) {
        const data = await response.json();
        setLinkedAccounts(data.data);
      } else {
        setError('Failed to fetch linked accounts');
      }
    } catch (err) {
      setError('An error occurred');
    } finally {
      setLoading(false);
    }
  };

  const handleLinkAccount = (provider) => {
    window.location.href = `http://localhost:3000/api/v1/auth/${provider}`;
  };

  const handleUnlinkAccount = async (provider) => {
    if (!window.confirm(`Are you sure you want to unlink ${provider}?`)) {
      return;
    }

    try {
      const response = await fetch(
        `http://localhost:3000/api/v1/auth/unlink-social/${provider}`,
        {
          method: 'DELETE',
          headers: { Authorization: `Bearer ${token}` }
        }
      );

      if (response.ok) {
        fetchLinkedAccounts();
      } else {
        const errorData = await response.json();
        setError(errorData.error?.message || 'Failed to unlink account');
      }
    } catch (err) {
      setError('An error occurred');
    }
  };

  if (loading) return <div>Loading...</div>;

  return (
    <div className="account-settings">
      <h2>Account Settings</h2>

      <section className="linked-accounts">
        <h3>Connected Social Accounts</h3>
        
        {error && <div className="error">{error}</div>}

        <div className="provider-list">
          {['google', 'facebook', 'github'].map((provider) => (
            <div key={provider} className="provider-card">
              <div className="provider-info">
                <h4>{provider.charAt(0).toUpperCase() + provider.slice(1)}</h4>
                {linkedAccounts?.linkedAccounts?.includes(provider) ? (
                  <>
                    <p className="status">✓ Connected</p>
                    <button
                      onClick={() => handleUnlinkAccount(provider)}
                      className="btn-unlink"
                    >
                      Disconnect
                    </button>
                  </>
                ) : (
                  <>
                    <p className="status">Not connected</p>
                    <button
                      onClick={() => handleLinkAccount(provider)}
                      className="btn-link"
                    >
                      Connect
                    </button>
                  </>
                )}
              </div>
            </div>
          ))}
        </div>
      </section>
    </div>
  );
}

export default AccountSettings;
```

### 3. Auth Store (Pinia)

```javascript
// stores/auth.js
import { defineStore } from 'pinia';
import { ref } from 'vue';

export const useAuthStore = defineStore('auth', () => {
  const user = ref(null);
  const token = ref(localStorage.getItem('accessToken'));
  const loading = ref(false);

  const setToken = (newToken) => {
    token.value = newToken;
    localStorage.setItem('accessToken', newToken);
  };

  const fetchUserProfile = async () => {
    if (!token.value) return;
    
    try {
      loading.value = true;
      const response = await fetch(
        'http://localhost:3000/api/v1/user/profile',
        {
          headers: { Authorization: `Bearer ${token.value}` }
        }
      );

      if (response.ok) {
        const data = await response.json();
        user.value = data.data;
      }
    } catch (error) {
      console.error('Failed to fetch user profile:', error);
    } finally {
      loading.value = false;
    }
  };

  const logout = () => {
    user.value = null;
    token.value = null;
    localStorage.removeItem('accessToken');
  };

  return {
    user,
    token,
    loading,
    setToken,
    fetchUserProfile,
    logout
  };
});
```

### 2. OAuth Callback Handler (Vanilla JS)

```javascript
// callback.js
class OAuthCallbackHandler {
  constructor() {
    this.init();
  }

  init() {
    const params = new URLSearchParams(window.location.search);
    const token = params.get('token');
    const error = params.get('error');

    if (error) {
      this.showError(decodeURIComponent(error));
    } else if (token) {
      this.handleToken(token);
    }
  }

  async handleToken(token) {
    try {
      localStorage.setItem('accessToken', token);
      await this.fetchUserProfile(token);
      window.location.href = '/dashboard';
    } catch (error) {
      this.showError('Failed to process login');
    }
  }

  async fetchUserProfile(token) {
    const response = await fetch(
      'http://localhost:3000/api/v1/user/profile',
      {
        headers: { Authorization: `Bearer ${token}` }
      }
    );

    if (!response.ok) {
      throw new Error('Failed to fetch user profile');
    }

    return response.json();
  }

  showError(message) {
    document.body.innerHTML = `
      <div class="error-container">
        <h2>Login Error</h2>
        <p>${message}</p>
        <a href="/login">Back to Login</a>
      </div>
    `;
  }
}
```

```javascript
// account-settings.js
class AccountSettings {
  constructor() {
    this.token = localStorage.getItem('accessToken');
    this.init();
  }

  async init() {
    await this.fetchLinkedAccounts();
    this.setupEventListeners();
  }

  async fetchLinkedAccounts() {
    try {
      const response = await fetch(
        'http://localhost:3000/api/v1/auth/linked-accounts',
        {
          headers: { Authorization: `Bearer ${this.token}` }
        }
      );

      if (response.ok) {
        const data = await response.json();
        this.linkedAccounts = data.data;
        this.render();
      }
    } catch (error) {
      console.error('Failed to fetch linked accounts:', error);
    }
  }

  setupEventListeners() {
    document.querySelectorAll('.btn-link').forEach((btn) => {
      btn.addEventListener('click', (e) => {
        const provider = e.target.dataset.provider;
        window.location.href = `http://localhost:3000/api/v1/auth/${provider}`;
      });
    });

    document.querySelectorAll('.btn-unlink').forEach((btn) => {
      btn.addEventListener('click', (e) => {
        const provider = e.target.dataset.provider;
        this.unlinkAccount(provider);
      });
    });
  }

  async unlinkAccount(provider) {
    if (!confirm(`Are you sure you want to unlink ${provider}?`)) return;

      const response = await fetch(
        `http://localhost:3000/api/v1/auth/unlink-social/${provider}`,
        {
          method: 'DELETE',
          headers: { Authorization: `Bearer ${this.token}` }
        }
      );

      if (response.ok) {
        await this.fetchLinkedAccounts();
      }
    } catch (error) {
      console.error('Failed to unlink account:', error);
    }
  }

  render() {
    const container = document.getElementById('providers-container');
    const linked = this.linkedAccounts.linkedAccounts || [];

    container.innerHTML = ['google', 'facebook', 'github']
      .map((provider) => `
        <div class="provider-card">
          <h4>${provider.charAt(0).toUpperCase() + provider.slice(1)}</h4>
          ${linked.includes(provider)
            ? `
              <p class="status">✓ Connected</p>
              <button class="btn-unlink" data-provider="${provider}">
                Disconnect
              </button>
            `
            : `
              <p class="status">Not connected</p>
              <button class="btn-link" data-provider="${provider}">
                Connect
              </button>
            `
          }
        </div>
      `)
      .join('');

    this.setupEventListeners();
  }
}

// Initialize
document.addEventListener('DOMContentLoaded', () => {
  new AccountSettings();
});
```

### 3. Token Management

```javascript
// Storage helper
class TokenManager {
  static setToken(token) {
    localStorage.setItem('accessToken', token);
  }

  static getToken() {
    return localStorage.getItem('accessToken');
  }

  static clearToken() {
    localStorage.removeItem('accessToken');
  }

  static isExpired() {
    const token = this.getToken();
    if (!token) return true;
    
    try {
      const payload = JSON.parse(atob(token.split('.')[1]));
      return payload.exp * 1000 < Date.now();
    } catch {
      return true;
    }
  }
}
```

## Troubleshooting

### Issue: Blank Page After OAuth Login

**Solution**: Ensure callback endpoint returns token as URL parameter:
```
/auth/callback?token=JWT_TOKEN
```

### Issue: CORS Error

**Solution**: Configure backend CORS to allow frontend origin:
```javascript
app.use(cors({
  origin: 'http://localhost:5173',
  credentials: true
}));
```

### Issue: Token Not Persisting

**Solution**: Ensure localStorage is available and not blocked:
```javascript
try {
  localStorage.setItem('test', 'test');
  localStorage.removeItem('test');
} catch (e) {
  // localStorage is not available
}
```

### Issue: Redirect Loop

**Solution**: Check if token is being properly validated before redirecting:
```javascript
if (token && validateToken(token)) {
  redirect('/dashboard');
}
```

## Best Practices

1. **Always validate tokens** before using them
2. **Store tokens securely** (consider httpOnly cookies for sensitive apps)
3. **Implement token refresh** for long-lived sessions
4. **Handle errors gracefully** with user-friendly messages
5. **Show loading states** during OAuth redirects
6. **Clear tokens on logout** from all storage locations
7. **Use HTTPS in production** for all OAuth communications
8. **Implement rate limiting** on OAuth endpoints
9. **Log OAuth events** for audit trails
10. **Test with multiple providers** before production

## Security Considerations

- ✅ Validate JWT tokens on the backend
- ✅ Use HTTPS for all OAuth communications
- ✅ Store sensitive credentials in environment variables
- ✅ Implement CSRF protection
- ✅ Validate state parameter in OAuth flow
- ✅ Use secure redirect URIs
- ✅ Implement session timeout
- ✅ Clear tokens on logout
